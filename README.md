# Manchester City Multilingual Hono Site

## Features
- Chinese / English / Spanish routes
- Hono + Cloudflare Workers
- JSON API
- Language auto-redirect on `/`

## Routes
- `/` auto redirect by browser language
- `/zh`
- `/en`
- `/es`
- `/zh/squad`
- `/en/squad`
- `/es/squad`
- `/zh/honours`
- `/en/honours`
- `/es/honours`
- `/api/club?lang=zh`
- `/api/club?lang=en`
- `/api/club?lang=es`

## Local dev
```bash
npm install
npm run dev
```

## Deploy
```bash
npm install
npm run deploy
```
